CruiseControl.NET : View Configuration Project Plugin
This page last changed on Sep 24, 2007 by rosspatterson.

The View Configuration Project Plugin shows the configuration for a project.

This plugin shows the entire configuration. If your configuration contains confidential information (e.g., passwords in <sourcecontrol> sections), you may not want to use it.

Example

<viewConfigurationProjectPlugin />

Configuration Elements

There is no configuration for this plugin.
